Typography

Typography is the art and technique of arranging type to make written language legible, readable, and appealing when displayed.

Key aspects:

Font selection

Choosing appropriate typefaces for different purposes.

Common categories include serif, sans-serif, monospace, cursive, fantasy .

Font pairing

Combining different fonts harmoniously.

Font size and readability

Ensuring text is easily readable.

Common guidelines include a minimum font size of 16px for body text.

Hierarchy

Using different sizes and styles to create a visual structure.

Color Theory

Color theory is the study of how colors interact and the psychological effects they have on viewers.

Important concepts:

Color wheel

A visual representation of color relationships.

Color schemes

Combinations like monochromatic, analogous, and complementary.

Color psychology

How different colors evoke specific emotions and associations.

Visual Hierarchy

Visual hierarchy is the arrangement of design elements in order of importance, guiding the viewer's attention through the content.

Techniques to establish hierarchy:

Size

Larger elements attract more attention.

Contrasting colors can emphasize important elements.

Using empty space to draw focus to specific areas.
